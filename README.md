# termuxcustom
Customize Termux Terminal


### New Features !

  - New customize color
  - New Fonts type


### Installation

for installation in termux :

```sh
$ git clone https://github.com/unlistedsquad/termuxcustom
```

```sh
$ cd termuxcustom
$ ls
$ sh setup.sh
```

### Customize

You can customize or change color and fonts.
For Color
``` sh
cd termuxcustom
ls
~/.termux/colors.sh
```
For fonts
```sh
cd termuxcustom
ls
~/.termux/fonts.sh
```
