# termuxcustom
Customize Termux Terminal


### New Features !

  - New customize color


### Installation

for installation in termux :

```sh
$ git clone https://github.com/unlistedsquad/termuxcustom
```

```sh
$ cd termuxcustom
$ ls
$ sh setup.sh
```

### Customize

You can customize or change color.
For Color
``` sh
cd termuxcustom
ls
~/.termux/colors.sh
```
